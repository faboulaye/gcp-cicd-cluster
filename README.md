# CI/CD cluster

Init terraform modules

```
npm run tf:init
```

Plan of terraform execution

```
npm run tf:plan
```

Execute terraform script

```
npm run tf:apply
```

Destroy terrafform deployment

```
npm run tf:destroy
```

To create jenkins namesapce in the CI/CD cluster

```
npm run ns:jenkins
```

Deploy jenkins on kubernetes cluster

```
npm run jenkins
```
